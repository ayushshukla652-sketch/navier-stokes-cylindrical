# Navier-Stokes Equations in Cylindrical Coordinates

**Author:** Ayush Shukla  
**Institution:** IIT Kanpur, Department of Mechanical Engineering

## About
This paper derives the Navier-Stokes equations for cylindrical 
coordinate systems from first principles and non-dimensionalises 
them using the Reynolds number.

## Topics Covered
- Kinematic relations between Cartesian and cylindrical coordinates
- Continuity equation derivation
- Radial, Azimuthal and Axial momentum equations
- Newton's law of viscosity for cylindrical systems
- Non-dimensionalisation using Reynolds number
- Boundary conditions for pipe flow
